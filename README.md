# GRUB2 THEME

##### Edit /etc/default/grub :
```shell
GRUB_THEME="/boot/grub/themes/EndeavourOS/theme.txt"
```
##### Update grub :
```shell
sudo grub-mkconfig -o /boot/grub/grub.cfg
```
